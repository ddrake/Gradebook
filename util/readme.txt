These are some helper scripts I've found to be useful.

The bash script 'gb' is intended to be placed in the top-level directory for the course.
It simplifies running the application for a given course and performs backup of data to the cloud
using rclone.  https://rclone.org/.  You will need to modify it to suit your purposes after installing
and configuring rclone on your system.

The folder 'webassign' contains some scripts for downloading scores from Webassign for import into
Gradebook.
